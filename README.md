# Self-diagnosis
매일 아침 상쾌하게 일어납시다.

## 사용법
1. [자가진단 프로그램 설치](https://github.com/1-EXON/Self-diagnosis/releases/download/v1.1/file.zip)
2. 자가진단 프로그램을 압축을 풀고 setting.txt에 이름,자신의 학교 이름,자신의 생일을 입력.
ex)홍길동,안녕중학교,071022
3. main.exe 실행
